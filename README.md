# altisenda-site

Marketing & support landing page for the [Altisenda](https://github.com/joseparrams/Altisenda) iOS app.

Served via **GitHub Pages** at:

- https://joseparrams.github.io/altisenda-site/

Used as the **Support URL** and **Marketing URL** on App Store Connect.

## Local preview

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

## Enabling GitHub Pages

Repo → **Settings → Pages** → *Source*: `Deploy from a branch` → *Branch*: `main` / `/ (root)` → **Save**.

The site publishes within ~1 minute.
